# AR EyeGlasses - Virtual Try-On

A high-performance Augmented Reality (AR) mobile application built with Unity and Vuforia Engine. This app allows users to try on various 3D eyewear models in real-time with precise control and a seamless user experience.

## Key Features

* **Real-Time AR Face Tracking:** High-fidelity eyewear placement using Vuforia's advanced face-tracking algorithms.
* **Dual-Layer Transformation System:**
    * **Quick Adjust:** Intuitive sliders for rapid scaling and positioning.
    * **Precise Tuning (+/-):** Dedicated fine-tuning buttons for millimeter-perfect alignment.
    * **Direct Input:** Input field support for specific coordinate values.
* **Dynamic In-App Gallery:**
    * Capture high-resolution snapshots with an integrated flash effect.
    * Browse saved captures in a native grid-based gallery.
    * Full-screen photo viewer with rotation support.
* **Modern HUD Design:** A sleek, animated slide-out HUD (Heads-Up Display) for a clean UI during the AR experience.

## Technical Stack

* **Engine:** Unity 2022.3 (LTS)
* **AR SDK:** Vuforia Engine 10.x
* **Graphics:** Universal Render Pipeline (URP)
* **UI Framework:** Unity UI (uGUI) & TextMeshPro
* **Scripting:** C# (Asynchronous Coroutines, File I/O, Event Systems)

## Installation and Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/GalipYilmaz/AR_Eye_Glasses.git
    ```
2.  **Open in Unity:** Use Unity Hub to open the project folder.
3.  **Setup Vuforia:** Ensure the Vuforia Engine package is imported via the Package Manager.
4.  **License Key:** Add your own Vuforia License Key in Window > Vuforia Configuration.
5.  **Build:** Switch platform to Android and click Build and Run.
